# Лабораторна робота №3
**Тема:** Наслідування: основи  
**Мета:** закріпити знання про базові класи, похідні класи, модифікатори доступу, використання `base`, поліморфізм.

## Виконання
- Створено базовий клас `User` з властивістю `Login` та кількістю виконаних операцій.
- Реалізовано похідні класи `Admin` та `Guest` з перевизначенням методів `DoOperation()` та `ShowInfo()`.
- Продемонстровано роботу **поліморфізму** на колекції користувачів.
- Додано метод для пошуку користувача з найбільшою активністю.
